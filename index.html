Financial Management System — First Time Setup

Follow these steps to set up your cloud database:

1. Go to console.firebase.google.com and create a free account

2. Create a new project (any name, e.g. "mdv-finance")

3. Go to Authentication → Enable Email/Password sign-in

4. Go to Firestore Database → Create database (Start in test mode)

5. Go to Project Settings → Scroll to "Your apps" → Add Web App → Copy the config below

Your data is stored in your own Firebase account. We do not have access to it.
